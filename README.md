# 🌤️ Weather Hub Dashboard

A lightweight, real-time weather tracking dashboard built to fulfill internship project milestones. This application communicates asynchronously with public weather APIs to deliver instant meteorological summaries based on user input.

- **Project Assignment:** Project #1 (Weather Hub Application)

### 🚀 Key Features
- **Live Search API Integrations:** Fetches current atmospheric data metrics seamlessly using OpenWeatherMap architecture.
- **Dynamic Condition Layouts:** The interface dynamically updates components depending on successful returns or error-state queries.
- **Clean Utility Styling:** Designed using an accelerated, modern dark mode palette courtesy of Tailwind CSS.
